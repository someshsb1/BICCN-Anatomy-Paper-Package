XYZ Anatomy Package

The XYZ Anatomy Package is the result of a community effort to curate anatomical data, tools, and use cases in open-access format.

This paper package is linked to previous efforts of the Brain Initiative Cell Census Network.

Linked Publications

Summaries and access to package articles with their specific contributions.

Anatomical Datasets

Comprehensive raw and processed brain data with metadata.

Analysis Tools

Tools for cell quantification, neuron tracing, and spatial registration.

Use Cases

Example applications and workflows using the datasets and tools.
